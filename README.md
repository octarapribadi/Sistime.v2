# Sistime
Sistem Informasi STMIK TIME atau Sistime, merupakan portal 
akademik yang memanajemen seluruh kegiatan akademik
civitas STMIK Time. Aplikasi ini dibuat menggunakan bahasa 
java, dengan menggunakan framework Java EE 8 sebagai backend,
dan primefaces sebagai frontend. Fungsi-fungsi Sistime hingga 
saat ini mencakup:
<ol type="1">
<li>Registrasi Mahasiswa</li>
<li>Kartu Rencana Studi</li>
<li>Kartu Hasil Studi</li>
<li>Penilaian Aktifitas Mahasiswa</li>
</ol>

# Spesifikasi
Sistime dijalankan dengan spesifikasi:
<ol type="1">
<li>JDK 8 atau diatasnya</li>
<li>Java Enterprise Edition 8</li>
</ol>
